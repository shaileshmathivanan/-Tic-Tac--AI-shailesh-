# -Tic-Tac--AI-shailesh-
Interactive Tic-Tac-Toe web app built with Python and Flask, featuring an AI opponent using Minimax and Alpha-Beta Pruning. The system ensures optimal gameplay and compares performance based on execution time and nodes explored through a simple, real-time user interface.
