  # 🎮 Tic-Tac-Toe AI (Minimax vs Alpha-Beta Pruning)

## 📌 Problem Description
This project implements an interactive Tic-Tac-Toe game where a human player competes against an AI opponent. The AI is designed to always make the optimal move using game theory algorithms. The system also compares the performance of two algorithms based on execution time and number of nodes explored.

---

## 🧠 Algorithms Used

### 1. Minimax Algorithm
Minimax is a decision-making algorithm used in two-player games. It explores all possible game states and selects the move that maximizes the AI's chances of winning while minimizing the opponent’s chances.

### 2. Alpha-Beta Pruning
Alpha-Beta Pruning is an optimization of the Minimax algorithm. It reduces the number of nodes evaluated by eliminating branches that do not affect the final decision, making the algorithm faster and more efficient.
